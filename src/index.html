Ejercicios JavaScript

  • Ejercicio 1. Concatenación de Cadenas:

    El ejercicio pide al usuario ingresar dos cadenas de texto y luego muestra la longitud de la cadena resultante al concatenarlas.

  • Ejercicio 2. Conversión de Celsius a Fahrenheit:

    Solicita al usuario una temperatura en grados Celsius y la convierte a grados Fahrenheit, mostrando el resultado.

  • Ejercicio 3. Cálculo del IMC:

    El ejercicio solicita dos números (peso y altura) y calcula el Índice de Masa Corporal (IMC), luego determina si el resultado es 'Obesidad de clase II'.

  • Ejercicio 4. Límites de Velocidad según Contaminación:

    Establece que en una carretera con límite de velocidad de 90 km/h, si la contaminación es mayor al 65%, el límite de velocidad debe ser 70 km/h. El algoritmo ajusta el límite según la condición de contaminación.

  • Ejercicio 5. Comportamiento del Semáforo:

    El ejercicio solicita al usuario el estado actual de un semáforo ('verde', 'amarillo intermitente', 'amarillo', 'rojo') y le indica si el coche debe detenerse o puede continuar, con una gestión de errores si el valor ingresado no es válido.

  • Ejercicio 6. Calificación en Letras:

    El algoritmo solicita una calificación numérica y devuelve la calificación correspondiente en formato de letras: Suspensa, Aprobada, Notable, Sobresaliente o Matrícula de Honor.

  • Ejercicio 7. Validación de Dominio de URL:

    Solicita una URL y extrae el dominio, indicando si es un dominio principal de 'Github' o un subdominio.

  • Ejercicio 8. Números Pares o Impares:

    Solicita un número y realiza un bucle desde 0 hasta ese número, imprimiendo si el número es par o impar.

  • Ejercicio 9. Ejercicio FizzBuzz:

    Solicita un número y realiza un bucle desde 1 hasta ese número. Imprime 'Fizz' si el número es múltiplo de 3, 'Buzz' si es múltiplo de 5, y 'FizzBuzz' si es múltiplo de ambos.

  • Ejercicio 10. Inversión de Número:

    Solicita un número al usuario y lo imprime al revés, mostrando cómo invertir el orden de sus dígitos.

  • Ejercicio 11. Validación de DNI Español:

    Solicita un número de DNI español y valida si es correcto según las reglas de formato y longitud del DNI español.

  • Ejercicio 12. Patrón de Triángulo en HTML:

    Solicita un número representando la altura de un triángulo y genera un patrón visual en HTML con asteriscos. El triángulo se mueve de izquierda a derecha según el ancho de la ventana.
